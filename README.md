# galgame—hero
**********************
## 剧情简介
#### 一名少年，沉睡了十年后突然苏醒，他会发现什么令人震惊的事实？
#### 这个世界运行的秩序是什么？传说中的英雄一族的真相又是什么？
#### 巨大的谜团，等待着你的探索！
**********************
## 实现的功能：
#### 1.存档和读档
#### 2.快进至选项
#### 3.清除对话框
#### 4.手动设置背景音乐的开闭
#### 5.在游戏社区发表评论
#### 6.有网络时，可以进行账号注册，并使用账号密码登录
#### 7.无网络时，可以不登录直接进行游戏，但此时无法访问游戏社区
